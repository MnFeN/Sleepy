# Sleepy

很简单的轻量级工具，可以通过编辑本地配置文件为 FF14 添加一些未启用的隐藏情感动作，例如随地睡觉、随地坐下等等。  
  
用户只需将该程序放置在游戏配置的 My Games 文件夹下的相应角色文件夹中，并以管理员模式运行即可。  
  
该工具仅会更改用户正常可访问的配置文件，不涉及读取内存、注入进程等操作。
