# Sleepy

很简单的轻量级工具，可以通过编辑本地配置文件为 FF14 添加一些未启用的隐藏情感动作，例如随地睡觉、随地坐下等等。  
  
**该工具仅会更改正常可直接编辑的用户配置文件，无需启动游戏进程，不涉及读写游戏内存、注入进程等操作。**   
原理：打开快捷栏配置文件，将指定的热键位置更改为情感动作。完全等价于打开文本编辑器手动修改几个数值。  

用户只需将该程序放置在游戏配置的 My Games 下相应角色文件夹中运行即可。
